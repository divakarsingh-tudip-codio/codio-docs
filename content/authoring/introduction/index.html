Codio Guides is a powerful authoring tool creating rich tutorial and instructional content as well as assessments. Here are a few of the key features.

  • Each page of content can control the layout of the IDE
  • Open and close files, terminal windows and html page previews
  • Launch commands from the content page and pipe terminal output into the content page
  • Create manually and automatically graded questions and assessments
  • Manage a table of contents
